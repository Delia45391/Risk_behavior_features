# Risk_behavior_features
This project consist in a dataset provided by kaggle and a set of questions to practice begginer concepts in MySQL and to train the logical thinking.

Based on the description from kaggle, “Risk behavior features” containing personal basic information and financial status, sourced from internet questionnaires. It covers key information such as age, gender, education level, marital status, employment status, income, credit score, and debt-to-income ratio, and can be used for machine learning research and learning scenarios. The dataset consists of approximately 80 records, faithfully obtained from questionnaire responses. Empty and missing rows have been removed.
